# Bakal

B2B prospecting automation service combining multi-channel outreach (Email + LinkedIn), AI-generated personalized copy, and automated performance optimization.

## Project Structure

```
bakal/
├── landing/                  # Client-facing marketing pages
│   ├── bakal-landing-page.html         # Dark theme
│   ├── bakal-landing-page-light.html   # Light theme
│   └── bakal-prequalification.html     # Eligibility form
├── app/                      # SaaS dashboard mockup
│   ├── bakal-saas-mockup.html          # Main dashboard UI
│   ├── campaigns-detail.js             # Campaign detail views
│   └── copy-editor.js                  # Copy & sequences editor
├── docs/                     # Documentation (prompt system, refinement loop)
├── CLAUDE.md                 # Full project context for Claude Code
└── README.md
```

## Tech Stack

| Component | Tool |
|-----------|------|
| Email/LinkedIn Automation | Lemlist |
| Workflow Automation | N8N |
| AI Generation | Claude API |
| Business Hub | Notion |
| Landing Page | Static HTML |

## Business Model

Agency-style done-for-you prospecting service at 350/month + 250 setup fee.
